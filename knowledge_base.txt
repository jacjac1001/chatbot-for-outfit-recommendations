The outfit recommendation chatbot suggests suitable clothing based on the user's gender and the weather condition.

The main purpose of the system is to help users quickly choose a simple outfit without manually searching through clothing images.

The system supports two gender categories: male and female.

The system supports two weather categories: hot and cold.

The system currently creates four recommendation labels: male_hot, male_cold, female_hot, and female_cold.

The system recommends one shirt, one pair of pants, and one pair of shoes for each outfit.

The chatbot works by reading the user's message, detecting gender, detecting weather, finding matching clothing images, and displaying an outfit recommendation.

The user should provide both gender and weather in the message for the best result.

A good example message is: I am male and it is hot today.

Another good example message is: I am female and the weather is cold.

If the user only provides gender but not weather, the chatbot should ask the user to provide hot or cold weather.

If the user only provides weather but not gender, the chatbot should ask the user to provide male or female.

If the user provides neither gender nor weather, the chatbot should ask for both gender and weather.

For hot weather, light and breathable clothing is recommended.

For hot weather, the outfit should feel comfortable, simple, and suitable for warm conditions.

For hot weather, thick jackets, heavy sweaters, and very warm clothing are not recommended.

For cold weather, warmer and layered clothing is recommended.

For cold weather, the outfit should help the user stay warm and comfortable.

For cold weather, light clothing alone may not be enough.

For male users in hot weather, a light shirt, comfortable pants, and breathable shoes are recommended.

For male users in cold weather, a warm shirt or hoodie, long pants, and closed shoes are recommended.

For female users in hot weather, a light top, comfortable pants or skirt, and breathable shoes are recommended.

For female users in cold weather, a sweater or warm top, long pants, and closed shoes are recommended.

The system selects clothing images from three main folders: Shirts, Pants, and Shoes.

The Shirts folder contains shirt or top images.

The Pants folder contains pants, trousers, shorts, or bottom clothing images.

The Shoes folder contains footwear images.

The system uses image filenames to find suitable clothing items.

A filename containing male_hot means the item is suitable for male users in hot weather.

A filename containing male_cold means the item is suitable for male users in cold weather.

A filename containing female_hot means the item is suitable for female users in hot weather.

A filename containing female_cold means the item is suitable for female users in cold weather.

The recommended filename format is label_number.extension.

Examples of good filenames are male_hot_001.jpg, male_cold_002.jpg, female_hot_003.jpg, and female_cold_004.jpg.

The system may fail to show an image if the matching folder does not contain a file with the correct label.

If a shirt image is missing, the user should add a matching image to the Shirts folder.

If a pants image is missing, the user should add a matching image to the Pants folder.

If a shoes image is missing, the user should add a matching image to the Shoes folder.
